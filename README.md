# Clinical Workflow Implementation Simulator

A healthcare workflow simulation system designed to model patient journeys across clinical stages.

## Features
- Patient admissions
- Workflow stage management
- Role-based access (Doctor / Nurse / Admin)
- Clinical notes per patient
- Audit logging
- Patient search and filtering
- Dashboard metrics
- Discharge workflow

## Tech Stack
- Python
- Streamlit
- Pandas

## Purpose
This project was built to simulate healthcare implementation workflows commonly seen in EHR/EPR systems.

It demonstrates:
- workflow configuration
- operational tracking
- audit trail logic
- patient lifecycle management
- clinical dashboarding

## Run Locally

```bash
python -m streamlit run app.py
```
